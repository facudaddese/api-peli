
# API de películas y series

Aplicación web de catálogo de películas y series construida con **React**. Permite explorar contenido mediante filtros por género y tipo, con búsqueda integrada y carga dinámica de datos desde **The Movie Database (TMDB)**. [Ver sitio](https://pelis-api.netlify.app/)

---

## Funcionalidades

-   Filtrado de contenido por **género**:
    -   Acción
    -   Terror
    -   Comedia
    -   Aventura
    -   Familiar
-   Filtrado por **tipo de contenido**:
    -   Películas
    -   Series
-  **Buscador integrado** para encontrar títulos específicos

---

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript (ES6+)
- React (Custom Hooks)
-  Fetch API & Async/Await
-    **TMDB API** (The Movie Database)
-   **React Spinners** (loader animado)


---

## API utilizada

Se consume la API pública de **The Movie Database (TMDB)** para obtener información de películas y series.

**Documentación oficial:**  
[TMDB API Documentation](https://developer.themoviedb.org/docs)
